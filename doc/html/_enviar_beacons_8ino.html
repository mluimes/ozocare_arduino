Detailed Description

Descripción breve de lo que hace este archivo.

Este archivo contiene el código principal de la aplicación Arduino, que lee valores de un sensor de gas y un sensor de temperatura, y transmite estos valores mediante un beacon.

void loop ( )

Función principal que se ejecuta en bucle.

◆ setup()

void setup ( )

Función de configuración inicial.

Se inicializa la comunicación serial, se configuran los sensores de gas y temperatura, y se establece la comunicación con el beacon. También se lee una vez los valores de gas y temperatura antes de comenzar la emisión del beacon.
